# ゲーム実装

## 国家部分

* 国家のパラメータ  
  * 領土：資源産出量に関与  
  * 人口：施設のレベル等を反映, 減らない  
  * 日付：戦闘時の環境決定に関与

* 管理するリソース  
  * 資金（税金と貿易で入手）  
  * 木材（事業と貿易で入手）  
  * 鉱石（事業と貿易で入手）  
  * 食料（事業と貿易で入手）

* サイクル  
1. 運営費決定＆建築＆貿易  
2. 時間経過  
* 攻撃戦  
* 防衛戦  
* 早送り

* 攻撃戦  
  * 任意のタイミングで派兵可能  
  * 戦闘開始まで移動期間を要する  
  * 出発時にメンバーと装備決定  
  * 途中帰還可能  
  * 帰還にも移動期間を要する

* 防衛戦  
  * ランダムに発生  
  * 国家に残っているキャラで応戦  
  * 国家に残っている装備で応戦

* 運営費決定  
  * 各施設には予算の割り振り項目がある  
  * 各割り振り項目には予算のminとMaxがある  
      
* 建築  
  * 建物の場所と種類固定  
    * 職業施設  
      * 戦士学校  
      * 魔法学校  
      * 教会  
    * 性格施設  
      * 図書館（蔵書内容）  
      * 広場（飾り付け）  
    * 精霊施設  
      * 精霊使役学校  
    * 武器制作施設  
  * レベルアップ  
  * 見た目変更可能（性格の偏り）  
  * 「時間経過」のステップに入るまでは計画段階のため取消可能

* 貿易  
  * 資金とそれ以外のリソースを相互に交換可能  
  * レートは各サイクルで変動  
    * 戦況反映  
    * 前シーズンの取引履歴反映

## 兵士

* ステータス  
  * パラメータ  
    * 各キャラクターに基礎値が設定されている  
    * 精霊によってバフがかかる  
      * 精霊使役学校（世界観：精霊の能力を引き出す）  
      * 国家反映度（世界観：対応する精霊国が発展することで精霊自体が強くなる）  
    * 各項目が異なる要素に影響を与える  
      * HP：0になると戦闘から離脱する  
      * CP：0になると精霊との契約が切れる  
      * STR：戦士のスキル効果に関与する  
      * INT：魔法使いのスキル効果に関与する  
      * FAI：聖職者のスキル効果に関与する  
      * AGI：移動速度に関与する  
  * 性格  
    * 各性格の発生確率はキャラによって固定（発生率0%の性格も含む）  
    * 施設によって性格の出る確率をあげる（後で正規化）  
    * 施設で性格分類の可能性が上がるとき、性格分類に関連した性格すべてが上がる  
  * ユニーク特性  
    * キャラ固有のもので、特定条件下でのパラメータ変動を発生させる  
  * 技  
    * 職業施設のレベルに応じて習得  
    * 戦闘時は武器に応じたものしか使えない  
    * 精霊リロールのタイミングで習得リセット  
  * 武器  
    * パラメータ上昇  
    * ランダムに提示された中から選択

* 精霊リロールによる休養・訓練期間  
  * 攻撃戦の場合は帰還からカウント

* 再出撃までの期間  
  * 攻撃戦の場合は帰還からカウント

# 戦闘部分

* サイクル	  
  * マップ決定  
    * 地形決定  
    * 天気ランダム決定  
  * バトル  
    * プレイヤー側の初期目標決め  
    * 敵側の初期目標決め  
    * オートバトル  
      * プレイヤーの援護魔法　  
  * リザルト  
    * 領土ゲットorロスト  
    * 資金ゲットorロスト  
    * 貿易レート変動

* マップ  
  * 地形  
    * 川：移動速度超低下  
    * 森：遠距離攻撃射程低下（視界不良）  
    * 沼：移動速度低下  
    * 平：ふつう  
    * 丘：遠距離攻撃射程向上（高い）  
    * 雪：移動速度低下  
  * 天候  
    * 晴れ：ふつう  
    * 降雨・降雪：遠距離攻撃射程低下（視界不良）  
    * 強風：特定の方向への移動速度Up, 逆方向の移動速度Down  
    * 猛暑：移動によって体力減少  
    * 極寒：時間経過で体力減少  
    * 雷雨：雨 \+ まれに雷に打たれてダメージを受ける  
  * 日付  
    * 1月：降雪、極寒、雪  
    * 2月：降雪、極寒、雪  
    * 3月：ふつう  
    * 4月：ふつう  
    * 5月：ふつう  
    * 6月：降雨、沼  
    * 7月：晴れ、森  
    * 8月：晴れ、猛暑、森  
    * 9月：晴れ、強風、森  
    * 10月：ふつう  
    * 11月：雷雨、強風  
    * 12月：降雪、極寒、雪  
  * 魔石（世界観：領土の境界を決める目印として魔石を設置しており、その周囲一定範囲が領土として扱われる。沖ノ鳥島と排他的経済水域の感じ）  
    * 相手の魔石を破壊すれば勝利  
    * ここから一定範囲内にキャラ配置可能

* 敵側の出撃メンバー決定（攻撃戦・防衛戦）  
  * デザイナーズ連携を作る  
  * 序盤は完全ランダムにキャラ選定  
  * 徐々にデザイナーズ連携を入れる  
    * 連携のお手本として連携を出すのもあり  
    * 開発の連携vsプレイヤーの連携みたいになるかも

* 敵側の出撃メンバー決定（模擬戦（PvP・Com戦共通））  
1. 両者全キャラに対して性格（精霊）ロール  
2. 両者全キャラに対してジョブロール  
3. 両者全キャラのパラメータロール（上方補正あり？）  
4. パラメータ準拠で上位3人を候補者に  
5. 候補者が被った場合はどちらのプレイヤーがそのキャラを候補に提示するかランダム抽選  
6. 抽選に外れたプレイヤーは下位キャラを繰り上げ当選  
7. それでも足りなければ余っているキャラを性格からリロールして配置。ただし性格候補上絶対になれないジョブしか余っていない場合は欠員で進行

* オートバトル  
  * 武器（職業）の基本行動指針がある  
  * 性格によって行動する  
  * プレイヤーによる援護魔法  
    * 地形変動  
    * 天候変化

    

* 基本行動指針  
  * 双剣  
    * 敵に近づいて攻撃  
  * 盾  
    * 味方にくっついて行動  
  * 杖  
    * 隠れて攻撃  
  * 魔導書  
    * 隠れていやがらせ  
  * 聖書  
    * 隠れて回復  
  * ロザリオ  
    * 隠れて支援

* 性格による影響  
  * 戦闘狂：何があっても敵への攻撃をやめない。  
  * 猪突猛進：敵は完全に無視して城へ攻撃する。  
  * 狡猾：敵の認識から外れて不意打ちを狙う。  
  * 目立ちたがり屋：敵の注意を引くことを優先する。  
  * 臆病者：敵の注意を集めないことを優先する。  
  * 献身的：敵と味方の斜線を遮るように移動する。  
  * 卑怯者：味方を盾にするように移動する。  
  * 熱血：攻撃を行いやすくなる。周囲のキャラも攻撃系の行動を行いやすくなる。  
  * 慎重：被ダメージを減らすための行動を取りやすい。  
  * 冷静：HPが危険状態になった場合に自主的に撤退する。  
  * 寂しがり：他のキャラと一緒に行動する。1人になると行動せず他のキャラを探し続ける。  
  * 一匹狼：他のキャラが交戦していない敵に優先して攻撃する。他のキャラが戦闘に参加すると別の敵のところへ向かう。  
  * クソ真面目：不意打ちを行わない。  
  * スケベ：異性のキャラの近くに行く。異性キャラの近くならステータスが上がるが、そうでなければステータスが下がる。  
  * 不思議ちゃん：行動が完全ランダム。  
  * おっちょこちょい：稀に味方を攻撃する。  
  * 怠け者：あまり行動しない。  
  * メンヘラ：自分がダメージを受けたとき、他のキャラがその攻撃を庇える状況にあった場合、庇われなかったことに逆ギレしてそのキャラを攻撃する。  
  * 天真爛漫：攻撃をあまり行わず、敵の周りをぐるぐる回っている。敵に狙われやすい。敵の攻撃を確率で回避する。  
  * 下世話：戦場に恋人関係の2人がいる場合、その近くに行く。近くにいる間、パラメータが大幅に上昇する。2人のうちどちらかが戦闘を離脱した場合、やる気をなくして自分も戦闘を離脱する。

* 技  
  * 戦士（近距離攻撃職）  
    * 双剣  
      * 斬撃：通常の斬撃。  
      * 隙の大きい強力な攻撃。  
      * 背後から攻撃することでダメージ倍率増加。  
      * 一定時間カウンター（回避 \+ 高倍率攻撃）準備状態になる。  
      * 不意打ち：視界不良エリアで戦闘中の場合、ダメージ倍率増加。  
    * 盾  
      * 通常の斬撃。  
      * 一定時間被ダメージ量を減少させる。  
      * 一定時間視野を広げる。  
      * 一定時間カバーできる距離が広がる。  
      * 一定時間ガードがジャストガードになる。  
  * 魔法使い（遠距離攻撃職・デバッファー）  
    * 杖  
      * 通常の攻撃。  
      * 長い詠唱の長射程の強力な攻撃。  
      * 認識されていない状態で攻撃するとダメージ倍率増加。  
      * 攻撃後に後退を行う、わずかにダメージ倍率の低い攻撃。  
      * 曲射により敵の注意を散らせる、ダメージ倍率の低い攻撃。  
    * 魔導書  
      * 通常の攻撃。  
      * 敵のSTRデバフ。  
      * 敵のINTデバフ。  
      * 敵のFAIデバフ。  
      * 敵のAGIデバフ。  
  * 聖職者（回復職）  
    * 聖書  
      * 通常の攻撃。  
      * 味方のSTRバフ。  
      * 味方のINTバフ。  
      * 味方のFAIバフ。  
      * 味方のAGIバフ。  
    * ロザリオ  
      * 通常の攻撃。  
      * 遠くの味方の体力を微回復する。  
      * 近くの味方に継続的に回復する加護をかける。  
      * 自分の周囲に回復エリアを展開する。  
      * 近くの相手にしか使えない非常に強力な回復。
